# 🏥 MediCare Plus — Doctor Clinic Website

A fully responsive, modern, and professional **Doctor Clinic Website** built with pure HTML, CSS, and Vanilla JavaScript. Designed to deliver a premium healthcare experience online.

## ✨ Features

- 🏠 **Multi-Page Layout** — Home, About, Services, Doctors, Appointment, Blog, Contact
- 📅 **Quick Appointment Booking** — Inline form on the hero section
- 🤖 **MediBot Chatbot** — AI-style chatbot assistant for patient queries
- 💬 **WhatsApp Integration** — Floating WhatsApp button for instant contact
- 📱 **Fully Responsive** — Works perfectly on mobile, tablet & desktop
- 🎨 **Premium Design** — Glassmorphism, gradient text, smooth animations
- ⭐ **Patient Testimonials** — Trust-building review section
- 🗺️ **Google Maps Embed** — Clinic location in the footer
- 🔢 **Animated Counters** — Stats bar with counting animations
- 🍔 **Mobile Hamburger Menu** — Smooth slide-in navigation for mobile

## 🚀 Getting Started (Run Locally)

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rajnish5821Kumar/Doctor_Website.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd Doctor_Website
   ```

3. **Open in browser:**
   - Simply open `index.html` in your browser, OR
   - Use **Live Server** extension in VS Code for best experience

---

## 🌍 Deploy on GitHub Pages

1. Go to your repository → **Settings** → **Pages**
2. Under **Source**, select `main` branch → `/ (root)` folder
3. Click **Save** — your site will be live at:
   ```
   https://rajnish5821kumar.github.io/Doctor_Website/
   ```
